# play.d2lang.com snippets mirror

This directory mirrors the current `Syntax snippets` from https://play.d2lang.com/.

- Source: `https://play.d2lang.com/js/snippets/snippets.js`
- Synced on: 2026-02-28
- File naming: `play_<snippet_name_slug>.d2`

Notes:
- These files are copied to preserve playground examples as-is.
- Some snippets use newer D2 syntax that `diago` does not fully parse yet.
- Therefore they are intentionally kept under `examples/play/` and are not part of `examples/*.d2` output generation.
